## Files Created/Modified

## Decisions Made

- **Both tasks committed together:** The .pen JSON format requires a complete valid document — writing partial state would produce an invalid file that Pencil cannot open. Both tasks were written atomically.
- **NavShell defined first:** Placed nav-shell before page frames in the nodes array so `ref` nodes in page frames have a defined target when the file is parsed.
- **Pricing Pro card spec followed exactly:** height 520 (Starter 460), borderTop brand-primary 4px, "Most Popular" badge in brand-primary pill — per locked plan spec.
- **Contact form error state inline:** Red text (#EF4444) below email field + disabled CTA at 40% opacity captured in contact frame children — no separate error state frame needed for this form.
- **Note nodes used for all annotations:** `type: "note"` nodes kept separate from frame children so they won't interfere with code generation or design screenshots.
