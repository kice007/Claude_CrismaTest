## Key Design Decisions

- **Reference sheet layout**: groups arranged vertically, each labeled with section header — not a page-sized frame
- **Toast left-border pattern**: 4px colored left border + matching icon distinguishes severity visually
- **Modal row pattern**: Desktop Dialog row (480px wide each) above Mobile Drawer row (375px, rounded-top corners, drag handle bar) — clearly shows responsive switching
- **Skeleton shades**: #E2E8F0 placeholder rects; circle for avatar, varying-width rects for text lines, pill for badge
- **EmptyState illustration**: brand-light bg frame with `users` icon in brand-primary — matches inline SVG spec from Phase 2
- **No stack traces on error pages**: explicitly shows clean user-facing design per not-found.tsx / error.tsx intent
