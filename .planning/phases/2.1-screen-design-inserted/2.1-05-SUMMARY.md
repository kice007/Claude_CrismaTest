## Key Design Decisions

- **2-col checkout layout**: 380px left column (plan summary, sticky) + flex-1 right column (form)
- **Billing toggle**: Monthly (active) / Annual tabs — annotation specifies live price switching $149/mo ↔ $119/mo
- **Stripe-style form**: Card number (full-width + card icon), Expiry/CVC (2-col row), Cardholder name; then Billing info section (email, address, city/postal row, country dropdown)
- **Subscribe CTA**: brand-primary full-width, 52px height; annotation notes 1.5s loading → /checkout/success
- **Success card**: 560px centered white card; brand-primary circle with check icon; 2 stacked CTAs (primary + outline)
- **No mobile frames**: per CONTEXT.md locked decision — responsive via Tailwind breakpoints
