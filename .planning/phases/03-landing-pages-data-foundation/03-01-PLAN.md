    - `RateEntry` interface: `{ count: number; expiresAt: number }`
    - Module-level `cache = new Map<string, RateEntry>()`
    - `setInterval` cleanup every 5 minutes — deletes entries where `now > entry.expiresAt`
    - Export `applyRateLimiter(req: NextRequest, maxRequests: number, windowSeconds: number): { allowed: boolean }`
    - IP extraction: `req.headers.get('x-forwarded-for')?.split(',')[0].trim() || req.headers.get('x-real-ip') || 'unknown'`
    - Logic: if no entry or expired → create entry with count=1, return allowed=true. If count < maxRequests → increment, return allowed=true. Otherwise return allowed=false.
